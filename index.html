Avoid using personal information :

Avoid using personal information such as your name, date of birth, or phone number in your password. This information is often easy for an attacker to obtain and use in a dictionary attack.

Use a unique password for each account :

It's important to use a unique password for each account. This way, if one password is compromised, the attacker will not have access to your other accounts.
